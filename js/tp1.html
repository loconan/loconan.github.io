Variables

Les variables ne sont accessibles que dans leur contexte courant ou par celui englobant le contexte courant. Les variables en JS n'ont aucun typage précis: elles sont déclarées grâce au mot-clé var.

Document Object Model

Analyse du script:

  • On crée un élément "div" en plus de celui déjà présent dans la page
  • On lui attribue un identifiant unique (div1)
  • On change la forme et la couleur de ses bordures
  • On l'affiche sur la page en l'ajoutant dans la div déjà existante

Pour sélectionner des éléments, on peut également utiliser la commande document.getElementByClassName(« ... »)[n°]

Pour ajouter du texte dans cette nouvelle div, on peut utiliser la commande: document.getElementById("div1").textContent = "....";

JS Objects

Pour créer des objets, on peut

  • Utiliser une fonction anonyme (Point)
  • Utiliser une fonction nommée (Voiture)
  • Utiliser un ensemble de variables (Personne)

Events

Une meilleure alternative pour lier une action à un évènement est de le faire dans le script JS avec: document.getElementById("...").addEventListener([event], [function], [false/true]);

Des exemples d'évènements sont:

  • keydown
  • keyup
  • mouseup
  • scroll
  • click
  • etc.
